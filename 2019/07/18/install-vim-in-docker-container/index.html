vi/vim的安装

vi就是简化版的vim，而且安装包只有1.4M，相比于nano的1.7M和vim的29.8M，真的是精简的很，而且常用的命令都可以使用，所以，推荐在container内部就使用vi(vim-tiny)就好了。

批量注释

  1. 按v进入virtual模式
  2. 选择要注释的内容
  3. 按control + v(mac) 或 ctrl + q(win)进入列模式
  4. 按大写I进入插入模式，输入注释#或者//
  5. 连续按两次ECS

批量删除注释

  1. ctrl+v进入块选择模式
  2. 选中每行开头的注释#或者//
  3. 按d进行删除

粘贴时取消自动注释
